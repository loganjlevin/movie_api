myFlix Application

This is the server-side component of a 'movies' web application that will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies. It will be created using Node.js, Express and PostgreSQL.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing description, genre, director, image URL, whether it's featured.
{
  title: "Citizen Kane",
  description: "Following the death of a publishing tycoon, news reporters scramble to discover the meaning of his final utterance."
  genre: "Drama"
  director: "Orson Welles"
  imageURL: "https://wikimedia.org/Citizen_Kane_poster.jpg"
  featured: "true"
}
GET data about a genre by name /genres/[Title] GET None A JSON object holding data about a genre containing a description
{
  title: "Drama",
  description: "Dramas are serious presentations or stories with settings or life situations that portray realistic characters in conflict with either themselves, others, or forces of nature."
}
Get data about a director /directors/[Name] GET None A JSON object holding data about a director containing bio, birth year, death year
{
  name: "Orson Welles",
  bio: "George Orson Welles was an American director, actor, producer, and screenwriter who is remembered for his innovative work in film, radio, and theatre. He is considered to be among the greatest and most influential filmmakers of all time."
  birthYear: "1915",
  deathYear: "1985",
}
Register a new user /users POST A JSON object holding data about the user to register
{
  username: "moviegoer1985",
  password: "password123"
}
A JSON object holding data about the user that was registered, including an ID.
{
  id: 5,
  username: "moviegoer1985",
  password: "password123"
}
Update a user's username by id /users/[ID]/[Username] PUT None A text message indicating whether the username was successfully updated.
Add a movie to a user's list of favorites /users/[ID]/favorites/[Title] PUT None A text message indicating whether the movie was successfully added to favorites.
Remove a movie from a user's list of favorites /users/[ID]/favorites/[Title] DELETE None A text message indicating whether the movie was successfully removed from favorites.
Deregister an existing user /users/[ID] DELETE None A text message indicating whether the user was successfully removed.
